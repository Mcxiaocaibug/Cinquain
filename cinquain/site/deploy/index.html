Runtime checklist

What must be green before release

  1. Ports 80 and 443 are open to the public internet.
  2. ./install.sh finishes without build or pull errors.
  3. https://your-domain/bootstrap creates the first administrator.
  4. ./doctor.sh passes landing, support, bootstrap, discovery, and client API checks.
  5. ./backup.sh creates an archive before every major upgrade.
