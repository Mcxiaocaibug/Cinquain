One-command Matrix deployment

Deploy Cinquain on your domain with one curated Docker stack.

Cinquain packages the Continuwuity core into a tighter self-hosting product. The default stack owns the database, TLS, reverse proxy, `.well-known`, and operator web surface so you do not have to assemble the network yourself first.

Open deploy guide Open operator support

Managed automatically

  • Embedded RocksDB persistence
  • Caddy-managed TLS
  • Client and federation discovery
  • Operator support pages

Deploy flow

Start the stack in three steps.

  1. Copy .env.example to .env and set your domain and operator email values.
  2. Run ./install.sh from the cinquain/ directory to pull the curated GHCR image and start the stack.
  3. Open /bootstrap, use the generated bootstrap secret, and create the first administrator.

Primary command

cd cinquain
cp .env.example .env
./install.sh

What you get

The default stack already includes the hard parts.

  • Caddy on ports 80 and 443
  • Homeserver routing and `.well-known` discovery
  • Persistent RocksDB and Caddy volumes
  • Backup, upgrade, and release helper scripts

Operator routes
