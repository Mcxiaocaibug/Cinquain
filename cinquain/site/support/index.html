Cinquain operator guide for this server.

This page is meant for the server operator after the stack is online. It complements the deployment scripts with a short checklist and verification targets.

Operator checklist

Bring the server to ready state

  1. Wait for TLS issuance to complete on first boot.
  2. Open /bootstrap and create the first administrator if first-run is still active.
  3. Run ./doctor.sh in the Cinquain deployment directory.
  4. Log in with the administrator account you created and confirm you can sync.
  5. Visit the `.well-known` endpoints and confirm discovery is live.

Common commands

./doctor.sh
./backup.sh
./upgrade.sh

Important constraint

The configured Matrix server name should be treated as permanent after first deployment. Changing it later is not a routine rename; it changes the identity surface of the homeserver.
